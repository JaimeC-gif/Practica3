#Projecte Prova: Documentació Oficial
**Benvingut a la documentació del projecte prova. Aquest lloc web ha estat generat automàticament mitjançant MkDocs i està preparat per ser desplegat a GitHub Pages.**

##Descripció del Projecte
Aquest projecte té com a objectiu establir una base de coneixement compartida i provar el flux de treball de desplegament continu per a documentació tècnica.

Característiques principals:
Markdown Nativa: Tota la documentació s'escriu en fitxers .md simples.

Velocitat: Generació de llocs estàtics ultra-ràpida.

Desplegament automàtic: Integració amb GitHub Pages mitjançant el comando gh-deploy.

##Instal·lació i Configuració
Per poder treballar en aquest projecte localment, necessites tenir instal·lat Python i MkDocs.
